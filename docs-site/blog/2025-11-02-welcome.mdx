# Welcome to HarnSpec!

We're excited to introduce HarnSpec - a lightweight, flexible Spec-Driven Development (SDD) methodology designed for modern AI-powered development.

{/* truncate */}

## What is HarnSpec?

HarnSpec is built on the principle that **AI coding agents need clear context without overwhelming documentation**. Traditional SDD approaches either lack the detail AI needs or create so much overhead that nobody maintains them.

## Key Features

- 🎯 **Write only what matters** - Clear intent AI can act on
- 🤖 **AI-native from day one** - Works with Cursor, GitHub Copilot, Aider, and custom agents
- 📈 **Scale naturally** - Solo dev → Small team → Enterprise
- 🔧 **Built-in workflow support** - Boards, timelines, dependency tracking
- 🎨 **Your structure** - Custom fields, flexible templates
