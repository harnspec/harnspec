# Specs

This directory contains HarnSpec specifications for this project.

## Quick Start

```bash
# Create a new spec
harnspec create my-feature

# List all specs
harnspec list

# View the board
harnspec board

# Validate specs
harnspec validate
```

## Structure

Each spec lives in a numbered directory with a `README.md` file:

```
├── 001-feature-name/
│   └── README.md
└── 002-another-feature/
    └── README.md
```

## Spec Status Values

- `draft` - Being authored or refined
- `planned` - Not yet started
- `in-progress` - Currently being worked on  
- `complete` - Finished
- `archived` - No longer relevant
