# HarnSpec Official Skills Registry

A collection of high-quality AI Agent skills for [HarnSpec](https://harnspec.github.io).

## Skills Catalog

| Skill | Description | Install Command |
| :--- | :--- | :--- |
| **harnspec** | Official SDD methodology skill for AI agents. | `npx skills add harnspec/skills@harnspec` |

## How to use

Ensure you have a HarnSpec-enabled project. To add a skill to your project:

```bash
npx skills add harnspec/skills@harnspec
```

This will download the skill to your `.agents/skills/` directory, allowing your AI agent (Claude Code, Cursor, etc.) to understand and interact with your project more effectively.

## Contribution

We welcome contributions to the official skills registry. Please follow these steps:

1. Fork the repository.
2. Create your skill in the `.agents/skills/` directory.
3. Ensure it follows the [HarnSpec Skill Standard](https://harnspec.github.io/docs/skills).
4. Submit a Pull Request.
